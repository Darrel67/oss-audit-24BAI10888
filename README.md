## Project Overview

This project is a structured audit of the **Linux Kernel**, conducted as part of the Open Source Software course.  

The objective of this audit is to:

- Study the origin and philosophy behind the Linux Kernel
- Analyze its licensing model (GPL v2)
- Examine its footprint on a Linux system
- Understand its ecosystem and community
- Compare it with a proprietary alternative
- Demonstrate practical Linux skills through shell scripting

The Linux Kernel was chosen because it forms the foundation of modern computing systems including servers, cloud infrastructure, Android devices, and embedded systems.

---

## About the Linux Kernel

The Linux Kernel is the core component of the GNU/Linux operating system. It manages:

- CPU scheduling
- Memory management
- Device drivers
- Process management
- System calls
- Hardware abstraction

It was created in 1991 by Linus Torvalds and is licensed under the GNU General Public License version 2 (GPL v2).
